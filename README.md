
# Instagram Clone


## Tech Stack🛠️

- Expo React-Native
- Firebase

## Steps to Run

To run the project,Run the following commands

- Fork the Repository:
- Create a new branch *dev-version*

Install the dependencies
```bash
  npm install
```
Run the application:
```bash
  npx expo start
```

#### Note:
- Install Expo Go on your device for developing the application.
